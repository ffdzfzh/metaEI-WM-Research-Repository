The antenna phase center was 61.5 cm from the metasurface center for all scans.
Each file contains a 128 x 128 matrix sampled over a 64 cm x 64 cm square
with 0.5 cm spacing.

- normal_incidence_61p5cm.mat: normal incidence.
- right_45deg_incidence_61p5cm.mat: 45-degree incidence from the observer's right when facing the metasurface.
- left_45deg_incidence_61p5cm.mat: 45-degree incidence from the observer's left when facing the metasurface.

Original filename mapping:

- 12-6-1.mat -> normal_incidence_61p5cm.mat
- 12-6-2.mat -> right_45deg_incidence_61p5cm.mat
- 12-11-1.mat -> left_45deg_incidence_61p5cm.mat

The original note referred to 12-5-2.mat for the right-incidence scan, but
the supplied file was named 12-6-2.mat. The supplied file was retained and
renamed; this discrepancy should be confirmed before publication.
