# Decision and Execution

This module converts predicted incident fields and task-level target patterns into discrete IMS control states. The released implementation includes far-field propagation-matrix construction, a modified Gerchberg-Saxton iteration, one-bit phase quantization, and hardware-oriented code mapping.

The upstream EM prediction outputs needed to construct arbitrary scene-specific dominant-ray maps are withheld in the pre-publication release. The included scan data allow the optimization stage to be inspected independently.
