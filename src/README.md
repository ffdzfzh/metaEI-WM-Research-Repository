# Core Research Modules

The uploadable source tree currently exposes two focused components of the manuscript pipeline:

- `perception/ims_localization_orb_demo/`: feature-based IMS identification and homography estimation using OpenCV ORB.
- `decision_execution/gs_algorithm/`: incident-field-aware Gerchberg-Saxton optimization and hardware code mapping.

The complete semantic EM prediction layer and end-to-end agent integration are not included in this pre-publication release. Their status is documented in [../docs/AVAILABILITY.md](../docs/AVAILABILITY.md).
