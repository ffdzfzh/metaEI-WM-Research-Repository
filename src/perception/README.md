# Perception and Representation

The manuscript perception module combines IMS pose estimation, CSI-based radio-source localization, autonomous exploration, RGB point-cloud reconstruction, semantic object extraction, and room topology recovery.

This repository releases only the standalone ORB-based IMS localization demonstration. TARE, R3LIVE, SpatialLM, RoomFormer, SpotFi, and the Linux 802.11n CSI Tool must be acquired from their upstream sources; they are not copied into this repository. See [../../third_party/README.md](../../third_party/README.md).
